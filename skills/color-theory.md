# 色彩理论 Skill

## Color System for Bilimind

### Semantic Colors
- **Primary (Emerald)**: #059669 — CTA buttons, active states, links
- **Knowledge (Blue)**: #3b82f6 — concept nodes, info panels
- **Learning (Purple)**: #8b5cf6 — method nodes, learning path
- **Success (Green)**: #22c55e — compilation complete, verified
- **Warning (Amber)**: #f59e0b — pending review, low confidence
- **Error (Red)**: #ef4444 — failed, rejected, errors
- **Highlight (Cyan)**: #06b6d4 — selected, focused

### Gradients
- Hero gradient: `linear-gradient(135deg, #059669, #06b6d4)`
- Card gradient: `linear-gradient(to bottom right, rgba(5,150,105,0.1), transparent)`
- 3D node glow: radial gradient canvas texture

### Accessibility
- Don't rely on color alone — add icons, text, or patterns
- Colorblind-safe palette (test with Coblis simulator)
- High contrast mode support
