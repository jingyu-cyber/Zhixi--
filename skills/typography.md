# 排版系统 Skill

## Typography for Bilimind

### Font Stack
```
--font-sans: 'Inter', system-ui, -apple-system, sans-serif;
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
--font-display: 'Inter', system-ui, sans-serif;  /* for headings */
```

### Type Scale (1.25 ratio)
- xs: 0.75rem (12px) — captions, metadata
- sm: 0.875rem (14px) — body small, labels
- base: 1rem (16px) — body text
- lg: 1.125rem (18px) — emphasized body
- xl: 1.25rem (20px) — card titles
- 2xl: 1.5rem (24px) — section headings
- 3xl: 1.875rem (30px) — page titles
- 4xl: 2.25rem (36px) — hero heading

### Line Height
- Headings: 1.2
- Body: 1.6
- Code: 1.5
- Small/captions: 1.4

### Font Weight
- Regular: 400 (body)
- Medium: 500 (buttons, labels)
- Semibold: 600 (card titles)
- Bold: 700 (headings)
- Extrabold: 800 (hero)

### Knowledge Node Typography
- Node name: 14px, semibold, truncate after 24 chars
- Definition: 13px, regular, line-clamp-3
- Difficulty dots: 10px colored circles
